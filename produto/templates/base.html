Funcionalidades

Este projeto, foi desenvolvido como seu total foco e objetivo: gerenciar o estoque de produtos, seja de um estabelecimento ou pessoa física.

  • O botão visualizar permite ver os atributos de cada item selecionado.
  • O botão editar permite editar quaisquer atributos do item selecionado.
  • O botão deletar permite deletar quaisquer atributos do item selecionado.
  • O botão adicionar permite adicionar um novo item.
  • O botão recarregar recarrega a página(útil para retirar os filtros ou buscas).
  • O botão baixar pdf gera uma tabela com todos os registro para download.
  • O botão baixar baixa a tabela selecionada como um arquivo pdf.
  • O botão voltar volta para a página anterior.
  • A paginação exibe 10 ítems por página.
